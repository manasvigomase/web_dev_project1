WEB DEVELOPMENT MODULE
7-Day Learning Journey
======================

DAY 1 – Basic HTML Tags
Learned fundamental tags for formatting and displaying content.
Tags: <p>, <b>, <i>, <h1>-<h6>, <marquee>, <br>, <hr>

DAY 2 – Media, Links & Tables
Embedded images, audio, video. Created hyperlinks and HTML tables.
Tags: <img>, <a>, <video>, <audio>, <table>, <tr>, <td>

DAY 3 – HTML Forms
Built interactive forms with various input types.
Tags: <form>, <input>, <textarea>, <select>, <button>

DAY 4 – Layout & CSS Selectors
Structured pages with semantic tags and styled with CSS selectors.
Tags: <header>, <nav>, <footer>
CSS: Class & ID selectors, Pseudo-classes, Combinators

DAY 5 – Box Model & Flexbox
Understood margin, border, padding, content. Built responsive layouts.
CSS: margin, padding, border, box-sizing, display:flex, justify-content, align-items

DAY 6 – JavaScript Basics
Added interactivity with variables, conditions, loops, functions, DOM.
JS: let/const/var, if/else, for/while, functions, DOM basics

DAY 7 – To-Do List Project
Built a fully functional To-Do List app using HTML, CSS & JavaScript.
Features: Add tasks, mark complete, delete tasks
